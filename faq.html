Часто задаваемые вопросы

Как зарегистрироваться в системе?
1. Перейдите на страницу входа
2. Нажмите кнопку "Регистрация"
3. Заполните все поля формы:
  - Имя пользователя (3-20 символов)
  - Email (будет использоваться для входа)
  - Пароль (минимум 6 символов)
  - Telegram тег (например: @username)
4. Примите условия использования
5. Нажмите "Зарегистрироваться"
6. Проверьте вашу почту для подтверждения
Как пополнить баланс?
На данный момент пополнение баланса возможно только через администратора системы.

Для пополнения:
1. Свяжитесь с администратором через Telegram: @FreskoWIN
2. Укажите сумму пополнения
3. Совершите перевод удобным способом
4. Администратор зачислит средства на ваш баланс

Минимальная сумма пополнения: 50 ₽
Максимальная сумма пополнения: 50,000 ₽
Как купить кейс?
1. Авторизуйтесь в системе
2. Перейдите в раздел "Кейсы"
3. Выберите понравившийся кейс
4. Нажмите кнопку "Купить кейс"
5. Подтвердите покупку в появившемся окне
6. Средства спишутся с вашего баланса
7. Вы получите предметы из кейса

Важно: Убедитесь, что на балансе достаточно средств для покупки.
Как участвовать в розыгрышах?
1. Авторизуйтесь в системе
2. Перейдите в раздел "Розыгрыши"
3. Выберите активный розыгрыш
4. Нажмите кнопку "Участвовать"
5. Дождитесь окончания розыгрыша
6. Победители выбираются автоматически

Правила:
- Один пользователь может участвовать в каждом розыгрыше только один раз
- Количество победителей указано в описании розыгрыша
- Результаты публикуются после окончания розыгрыша
Что делать, если забыл пароль?
Если вы забыли пароль от своей учетной записи:

1. На странице входа нажмите "Забыли пароль?" (функция в разработке)
2. Или свяжитесь с администратором через Telegram: @FreskoWIN
3. Предоставьте информацию для идентификации:
  - Email учетной записи
  - Имя пользователя
  - Telegram тег
4. Администратор сбросит ваш пароль и отправит новый

Рекомендация: После восстановления доступа смените пароль на новый.
Как стать администратором?
Статус администратора назначается только владельцем системы.

Чтобы получить права администратора:
1. Свяжитесь с владельцем через Telegram: @FreskoWIN
2. Объясните, зачем вам нужны права администратора
3. Пройдите проверку
4. Получите доступ к админ-панели

Обязанности администратора:
- Управление пользователями
- Настройка системы
- Проведение розыгрышей
- Поддержка пользователей
- Модерация контента
Почему не работает фоновая музыка?
Фоновая музыка может не работать по нескольким причинам:

1. Настройки браузера: Некоторые браузеры блокируют автовоспроизведение
  Решение: Нажмите на любую область страницы для разблокировки

2. Настройки звука: Музыка может быть отключена
  Решение: Нажмите на иконку звука в правом верхнем углу

3. Проблемы с файлом: Файл музыки может быть поврежден
  Решение: Сообщите администратору о проблеме

4. Блокировка рекламы: Некоторые блокировщики могут мешать
  Решение: Добавьте сайт в исключения
